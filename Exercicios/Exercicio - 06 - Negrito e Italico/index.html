Principais formatações utilizadas (Negrito e Italico)

Negrito/Destaque

Nesta frase iremos utilizar a TAG "B" para deixar a palavra/frase em negrito, porém ela não é semantica.

Agora iremos utilizar uma tag semantica para deixar a palavra em negrito, cujo a mesma é a TAG "Strong"

Italico/Enfase

Nesta frase iremos utilizar a TAG "I" para deixar um termo em itálico, porém a tag I não é semantica.

Agora vamos utilizar a tag semantica para deixar o termo em itálico, cujo a mesma é a TAG "em"
