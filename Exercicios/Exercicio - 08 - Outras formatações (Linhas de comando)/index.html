Outras formatações

Código-fonte

O comando document.getElementById('teste') é escrito em uma linguagem chamada JavaScript

Vamos utilizar a tag "CODE" para deixar o comando com uma cara mais apropriada para linhas de comandos.

Exemplo: document.getElementById('teste')

Pré-Formatação

Agora vamos utilizar uma tag "PRE" junto com a tag "Code" para deixar uma linha de código formatada, no estilo que precisa ser configurada.

Exemplo:

            
        num = int(input('Digite um número'))
        if num % 2 == 0:
            print (f'O número {num} é PAR')
        else:
            print(f'O número {num} é ÍMPAR')
        print('Fim do programa')

Citações simples

Vamos utilizar a TAG ('Q') para citar uma frase, colocando "" (ASPAS) entre o texto aplicado

Como diria o pai de um amigo: O computador é um burro muito rápido
